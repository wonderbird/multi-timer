# Feature Requests

## Audio Reminders

Users want to configure audio reminders for their timers. They should be able to specify an audio file and a time interval before the timer ends. When the timer reaches that interval, the specified audio file will play as a reminder, such as "one more minute to go."

## Visual Notifications

Users want to disable all audio notifications and instead have the screen flash. This feature helps meditation teachers give instructions at the correct time without disturbing the meditating group with sounds and spoken texts.
