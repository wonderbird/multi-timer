# TestFlight Deployment Guide

Deployment to TestFlight follows the instructions in [Build and release an iOS app](https://docs.flutter.dev/deployment/deployment/ios.md) [google2025aj].

This guide documents necessary considerations for a Flutter App. Thus, read this gide first before following the official instructions.

Additional resources are shown at the bottom of this page.

## App Configuration

- **Bundle ID**: `systems.boos.multiTimer`
- **App Name**: "Multi Timer für Atempraxis"
- **Display Name**: "Multi Timer"
- **Primary Category**: Health & Fitness
- **SKU**: `multi-timer-001`
- **iOS Deployment Target**: iOS 26.0

The `.txt` files in this folder contain the documentation submitted to App Store Connect [appleinc2023b].
